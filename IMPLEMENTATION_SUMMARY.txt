================================================================================
                          TROUBLESHOOTING
================================================================================

ISSUE: "Cannot POST /api/chat"
CAUSE: Frontend not reaching backend
FIX:   1. Check .env has correct REACT_APP_API_URL
       2. Verify it includes https:// (not http://)
       3. Verify it includes -5000 port suffix
       4. Restart frontend after changing .env
       5. Check backend is running

ISSUE: "CORS error" in browser
CAUSE: Backend not accepting frontend domain
FIX:   1. Verify backend is running
       2. Check REACT_APP_API_URL is correct
       3. Verify domain ends with .app.github.dev or is localhost
       4. Restart backend
       5. Try different browser

ISSUE: Frontend shows blank screen
CAUSE: JavaScript error or loading issue
FIX:   1. Open DevTools (F12)
       2. Check Console tab for errors
       3. Check Network tab - any failed requests?
       4. Clear cache (Ctrl+Shift+Delete)
       5. Hard refresh (Ctrl+F5)

ISSUE: Servers won't start
CAUSE: Port already in use or dependencies missing
FIX:   1. Kill other processes: lsof -i :5000 / lsof -i :3000
       2. Check npm is installed: npm --version
       3. Reinstall deps: rm -rf node_modules && npm install
       4. Check .env is correct: cat .env

For more issues, see QUICK_START.md or VALIDATION_CHECKLIST.md
